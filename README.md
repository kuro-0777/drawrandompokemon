
# Draw Random Pokemon

A small drawing game to play with friends: guess and draw randomly selected Pokémon using a simple board and tools.

## Features
- Interactive drawing board with basic brush/erase tools
- Undo / redo support
- Hints to help guessing
- Save drawings as images
- Clean, responsive layout
